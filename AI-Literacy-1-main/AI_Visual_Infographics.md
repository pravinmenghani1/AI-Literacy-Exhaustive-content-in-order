## 🧠 Infographic 7: Neural Network Architecture

```
🧠 FROM BIOLOGY TO TECHNOLOGY 🧠

     │                                      │
     ▼                                      ▼
🌿 Dendrites ──► 🔵 Cell Body ──► 🌿 Axon    📊 Inputs ──► ⚙️ Aggregator ──► 📈 Output
     │              │              │           │              │                │
     ▼              ▼              ▼           ▼              ▼                ▼
Receive signals  Process info   Send signals  Weighted sum   Activation      Final result
from other      & make         to other      of inputs      function        to next layer
neurons         decisions      neurons       + bias         (non-linear)    or output

🔗 NEUROPLASTICITY           🔗 LEARNING PROCESS

or weaken based on use      to minimize prediction errors

🏗️ NETWORK STRUCTURE:
Input Layer ──► Hidden Layer(s) ──► Output Layer
    │               │                   │
    ▼               ▼                   ▼
Raw data        Pattern detection    Final prediction
(Age, Salary)   (Complex relationships) (Loan approval)
```
